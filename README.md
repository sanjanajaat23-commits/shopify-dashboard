# Shopify Analytics Dashboard

Modern Shopify analytics dashboard built using Next.js and Tailwind CSS.

## Features
- Revenue analytics
- Orders tracking
- Customer statistics
- Responsive UI
- Modern dashboard UI

## Tech Stack
- Next.js
- React.js
- Tailwind CSS
- TypeScript
